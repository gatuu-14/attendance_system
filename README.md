Student Attendance System
A simple web-based Student Attendance System built with Django that allows teachers to register students, mark attendance, and generate reports.

Features
Register Students - Add new students with complete details

Mark Attendance - Mark daily attendance as Present, Absent, or Late

Generate Reports - View attendance reports for any date range

Student List - View all registered students

Student Details - View individual student profile and attendance history

Dashboard - View attendance statistics and charts

Print Reports - Print attendance reports directly from browser

Tech Stack
Backend: Django (Python)

Database: SQLite (default)
Installation
Prerequisites
Python 3.8 or higher

pip (Python package manager)

Frontend: Bootstrap 5, Chart.js

Icons: Font Awesome
